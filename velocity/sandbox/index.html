How To Set Up CamSpeed

1. Calibration (The most critical step!): Put an object of known length (like a 30cm ruler or a meter stick) on camera. Drag the green handles over it, set the exact distance value in the menu, and confirm.

2. Gate Mode: Best for high speed timing. Position Gate 1 and Gate 2. When an object passes through Gate 1 and then Gate 2, the clock calculates velocity based on elapsed interval.

3. HSV Color Track Mode: Click anywhere directly on the video feed to sample a color. Adjust match tolerance and tracking filters to trace a path continuously.
